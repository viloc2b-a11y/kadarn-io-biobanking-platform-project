# Kadarn Governance & Compliance Pack
## Version 1.0 — v1.0.0-beta

This directory contains the governance and compliance documentation for the Kadarn platform. These documents map Kadarn's architecture, data model, and operational controls to regulatory frameworks including HIPAA, GDPR, SOC 2, ISO 27001, ISO 27701, 21 CFR Part 11, and ISO 20387.
