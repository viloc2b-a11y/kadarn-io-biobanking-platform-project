**Part 5: Commercial Value (3 min)**
- Sponsors: find capable sites in minutes, not months
- Sites: demonstrate capabilities objectively
- Kadarn: evidence infrastructure, not a marketplace

## 5. Internal User Guides

### Site Director Quick Start
1. Log in to Kadarn
2. Upload documents via Discovery
3. Wait for processing (pipeline runs automatically)
4. Review Recognition Dashboard
5. Check Sponsor Readiness
6. Review incoming Opportunity Briefs
7. Grant/decline consent
8. Access Feasibility Passport when revealed

### Sponsor Quick Start
1. Log in to Kadarn
2. Open Sponsor Search
3. Enter capability requirements
4. Review anonymous matches
5. Create Discovery Workspace with protocol details
6. Generate Opportunity Brief
7. Wait for institution response
8. Upon consent, identities revealed
9. Access Living Feasibility Passport

### Kadarn Internal Reviewer Quick Start
1. Log in with internal credentials
2. Access Visibility Inspector to audit policies
3. Check Firewall Status for evidence quality
4. Review Identity Resolution status
5. Monitor Connector Health for all 7 providers
6. Generate audit reports as needed
