## Value Creation

Kadarn creates value by solving a structural inefficiency in the biospecimen and clinical research industry: **the gap between what institutions can demonstrate and what sponsors need to verify.**

### Value for Institutions

| Before Kadarn | With Kadarn |
|---|---|
| Send PDFs and emails to prove capability for each sponsor inquiry | Build one evidence profile that serves all sponsors |
| No standardized way to demonstrate readiness | Program Readiness — a common language for capability |
| Manual, reactive compliance | Continuous, evidence-backed readiness |
| Reputation-based selection favors established players | Evidence-based discovery enables any institution to compete |
| No visibility into gaps or improvement path | Actionable gap analysis and prioritized recommendations |

The core value for institutions: **convert operational investment into discoverable, verifiable capability.**

### Value for Sponsors

| Before Kadarn | With Kadarn |
|---|---|
| 6-8 weeks manual site qualification per program | Evidence-backed discovery in minutes |
| Scattered documents across email, portals, PDFs | Structured, comparable capability profiles |
| Trust based on reputation and past relationships | Trust based on verifiable, current evidence |
| No ongoing visibility into institution readiness | Portfolio monitoring with proactive alerts |
| Risk of selecting under-qualified sites | Evidence trail enables informed risk assessment |

The core value for sponsors: **replace manual qualification with evidence-backed discovery.**

### Value for CROs

| Before Kadarn | With Kadarn |
|---|---|
| Manual site identification and qualification | Multi-program portfolio with readiness visibility |
| No standardized comparison across sites | Capability Matrix enables side-by-side evaluation |
| Reactive monitoring | Proactive alerts for readiness changes |

### Value for Biobanks

| Before Kadarn | With Kadarn |
|---|---|
| Capabilities described in static catalogs | Dynamic capability profiles backed by evidence |
| No standardized way to demonstrate collection/processing/storage quality | Program Readiness for biobanking-specific capabilities |

### Value for Regulators

| Before Kadarn | With Kadarn |
|---|---|
| Reactive audit preparation | Continuous evidence maintenance |
| Manual evidence collection for inspections | Audit-ready evidence trails with provenance |

## Value Capture

Kadarn captures value through a multi-tier model aligned with the value each actor receives.

### Pricing Philosophy

Kadarn's pricing is based on three principles:

1. **Value-aligned**: Actors who derive more value pay more. Sponsors who discover institutions through Kadarn capture more value than institutions who build a profile.
2. **Usage-correlated**: Pricing scales with platform usage, not arbitrary tiers. More programs, more institutions, more evidence → more value → higher price.
3. **Transparent**: No hidden fees. No per-transaction charges that create misaligned incentives. Kadarn charges for access to the platform and its intelligence, not for transactions between parties.

### Revenue Streams

| Stream | Who Pays | What They Get | When |
|---|---|---|---|
| **Institution Subscription** | Institutions (labs, hospitals, biobanks) | Readiness evaluation, evidence management, capability workspace, gap analysis, improvement recommendations | From day one |
| **Sponsor Subscription** | Sponsors (pharma, biotech, diagnostics) | Institution discovery, program matching, portfolio monitoring, evidence review, comparison tools, exportable reports | Phase 2 (Sponsor Activation) |
| **CRO Subscription** | Contract Research Organizations | Multi-program portfolio, cross-sponsor visibility, capacity assessment, monitoring | Phase 3 (Network Growth) |
| **Network License** | Large sponsors, CROs, networks | Unlimited programs, API access, custom program types, dedicated support | Phase 4+ |
| **Certified Engine Access** | All actors (optional add-on) | Compliance Engine, Risk Engine, AI-assisted matching (future) | Phase 5 |

### Why No Per-Transaction Fees

Many marketplaces charge per transaction — a percentage of each deal. Kadarn deliberately avoids this model.

Per-transaction fees create misaligned incentives:
- The platform is incentivized to push transactions, even if the match isn't ideal
- Sponsors and institutions are incentivized to circumvent the platform to avoid fees
- The platform's revenue depends on transaction volume, not evidence quality

Kadarn's subscription model aligns incentives:
- Kadarn is incentivized to improve evidence quality — better evidence → more sponsor trust → more subscriptions
- Sponsors and institutions are incentivized to use the platform fully — they've already paid
- Revenue grows with platform value, not transaction volume

## Buyer Personas

### Institution Buyer
**Who**: Lab Director, Research Operations Manager, Biobank Director
**Pain**: Spending weeks responding to sponsor RFPs with the same evidence over and over. No way to proactively demonstrate readiness.
**Decision trigger**: A missed program opportunity because they couldn't prove readiness fast enough.
**Budget source**: Operations budget, research infrastructure budget.
**ROI**: Time saved on RFP responses + new program opportunities from sponsor discovery.

### Sponsor Buyer
**Who**: Clinical Operations Director, Program Manager, Outsourcing Manager
**Pain**: 6-8 week site qualification cycles. Inconsistent evidence quality. No ongoing visibility into site readiness.
**Decision trigger**: A program delay caused by slow site qualification.
**Budget source**: Clinical operations budget, outsourcing budget.
**ROI**: Faster program startup + reduced risk of site failure + ongoing portfolio visibility.

### CRO Buyer
**Who**: VP of Clinical Operations, Site Selection Lead
**Pain**: Managing site qualification across multiple sponsors and programs with no unified view.
**Decision trigger**: Losing a sponsor because site selection took too long.
**Budget source**: Operations budget, technology budget.
**ROI**: Multi-program efficiency + faster sponsor response + differentiated service offering.

## Adoption Strategy

### Phase 1: Institution-Led Growth (Current)
- Target: 10-20 early adopter institutions (academic medical centers, reference labs, biobanks)
- Value proposition: "Build your readiness profile once. Serve all sponsors."
- Acquisition: Direct outreach, industry conferences, existing relationships
- Success metric: Institutions reaching READY status and publishing to network

### Phase 2: Sponsor Pull (3-6 months)
- Target: 3-5 sponsor organizations with active programs
- Value proposition: "Discover ready institutions in minutes, not weeks."
- Acquisition: Warm introductions from early adopter institutions, industry partnerships
- Success metric: Sponsors discovering and selecting institutions through Kadarn

### Phase 3: Network Flywheel (6-12 months)
- More institutions → more sponsor interest → more programs → more institutions
- CRO and Biobank adoption as the network grows
- Success metric: Unsolicited inbound interest from both institutions and sponsors

### Phase 4: Market Expansion (12-18 months)
- Marketplace launch as consumption layer
- Industry-standard program types
- Geographic expansion
- Success metric: Kadarn readiness status recognized as industry signal

---

## Expansion Strategy

Kadarn's expansion follows the evidence:

1. **Program Type Expansion**: From 3 initial readiness programs (Biospecimen Collection, PBMC Processing, IVD Validation) to coverage of all major biospecimen and clinical research domains
2. **Geographic Expansion**: From US-focused to global, adding region-specific regulatory evidence requirements
3. **Vertical Expansion**: From biospecimen/IVD to adjacent domains — clinical trials, real-world evidence, genomics
4. **Ecosystem Expansion**: Certified Engines (Compliance, Risk, Quality) as add-on layers

Each expansion phase is gated by evidence: Kadarn enters a new domain only when the evidence taxonomy and program types for that domain are mature enough to support trustworthy readiness assessment.
